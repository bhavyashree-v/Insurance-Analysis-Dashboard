# Insurance-Analysis-Dashboard

## Overview

Developed an Insurance Analytics Dashboard in Power BI to analyze policy claims, approval trends, customer demographics, and policy expiration risks.

## Tools Used - 
* Power BI
* SQL
* DAX
* Power Query

## Key Insights
* Analyzed Rs. 251M total claims
* Identified 34% claim approval rate
* Flagged 453 expiring policies as revenue risk
* 46+ age group had highest policy concentration

## Features - 
* Claims Analysis
* Policy Expiration Tracking
* Customer Demographic Analysis
* Interactive KPI Dashboard
